## Introduction to the Redux Challenges

[Redux](https://redux.js.org/) 是 JavaScript 状态容器，提供可预测化的状态管理。Redux 可以让你构建一致化的应用，运行于不同的环境（客户端、服务器、原生应用），并且易于测试。Redux 支持很多界面库，本章是在介绍其和 React 组合前的一个入门。
